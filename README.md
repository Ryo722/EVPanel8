# EVPanel8

スライドパズル × マッチ3 のブラウザパズルゲーム。

2005年頃に「ポケモンだいすきクラブ」で公開されていた Flash 製ミニゲーム「イーブイパネル8」にインスパイアされた、オリジナルの再構築作品です。

## 遊び方

1. 属性（💧水 / 🔥炎 / ⚡雷）を選んでゲーム開始
2. 3×3 の盤面でパネルをスライドし、同じ属性を3枚揃えて消す
3. 選んだ属性を多く消すと進化ゲージが上昇し、ボーナス獲得
4. フィーバー属性を消すと制限時間が増える
5. 制限時間（3分）内にハイスコアを目指す

## 技術スタック

- Vite + React + TypeScript
- CSS Modules
- Vitest
- GitHub Pages

## 開発

```bash
npm install
npm run dev      # 開発サーバー起動
npm run build    # プロダクションビルド
npx vitest run   # テスト実行
```
